# 更新日志

### 本分支特性

本分支由 Jingshiro 维护，项目地址：[https://github.com/Jingshiro/legado](https://github.com/Jingshiro/legado)

- **详细阅读记录**：记录更细致的阅读信息，支持导出与统计。
- **AI 助手**：引入大模型交互能力，支持工具调用，AI 可查询和管理书架、书源、订阅等数据。支持 AI Agent 通过 HTTP 调用，可以将此skills发送给你的 AI Agent 使用：`https://pub-0e6ce5d0161d4148a621c594405613f1.r2.dev/reading-profile-skill.zip` 
- **想法分享与导出**：长按文字可写下想法，可生成分享卡片，也可一键导出至 Obsidian。
- **阅读小票**：可在书籍首页和尾页显示小票，录入书籍的评分和阅读时长。
- **WebDAV 备份管理增强**：支持在本地对云端备份进行删除、重命名等操作。
- **主题全量导出功能**：支持一键分享全部主题设置。
- **读完/刷书标签**：记录书籍阅读状态（读完/N刷），并在书架上进行展示。

**2026/05 - UI 全面迭代为 Material Design 3**

- 整体 UI 采用 MD3 设计语言重构，覆盖书架、书籍详情、阅读页、弹窗等所有核心页面
- 书架页面改造为 MaterialCardView 12dp 圆角卡片布局
- 全面 MD3 颜色系统迁移，所有页面适配新配色
- AI 助手界面优化：白色按钮、磨砂半透明底栏、统一标题栏配色

**2026/04 - 知识管理与备份增强**

- 新增导出想法到 Obsidian 功能，支持 REST API 和本地文件两种模式
- WebDAV 备份管理增强：支持在本地对云端备份进行删除、重命名等操作
- 新增主题管理对话框，支持主题的导入、导出和删除
- 阅读小票更新为藏书票结算清单样式，优化排版
- CI/CD 优化：检查更新源切换至 Gitee，R2 自动清理仅保留最近 10 个版本

**2026/03 - AI 助手完善**

- AI 助手新增独立入口，在"我的"页面可直接与 AI 对话
- AI 助手新增工具调用功能：
  - 只读工具：查询书架、搜索书籍、查看书源/订阅源、获取阅读统计等
  - 写操作工具：修改书籍分组、启用/禁用书源和订阅源等（需用户确认）
- 修复 AI 助手工具调用轮次过多时的超时异常，上限提升至 90 轮
- 内置查看详细阅读记录功能
- 优化代码，修复问题
